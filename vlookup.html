VLOOKUP Tutorial

What is VLOOKUP?

VLOOKUP is a traditional Excel formula used to search for a value in the first column of a table and return a corresponding result from another column in the same row.

Why Use VLOOKUP?

✔ Simple to use

✔ Widely available

✔ Good for basic lookups

Example

find the details of the employee ID

=VLOOKUP(C17,Table13, {2,3,4},0)
