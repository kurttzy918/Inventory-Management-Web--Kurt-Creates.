📊 VLOOKUP Tutorial

📌 What is VLOOKUP?

VLOOKUP is a traditional Excel formula used to search for a value in the first column of a table and return a corresponding result from another column in the same row. It's one of the most iconic lookup functions, perfect for vertical data alignment.

⚡ Why Use VLOOKUP?

✔ Simple syntax & widely compatible (Excel 2007+)

✔ Fast for small to medium datasets

✔ Great for merging data from two tables

✔ Reliable exact or approximate matches

📁 Real Example

Scenario: Find employee details based on ID from a table.

=VLOOKUP(C17, Table13, {2,3,4}, 0)

➡️ Searches for value in C17 within first column of Table13, returns columns 2,3,4 (department, name, salary) with exact match.

💡 Pro tip: Use 0 (FALSE) for exact match, 1 for approximate range.

🧠 VLOOKUP vs XLOOKUP

VLOOKUP is limited to rightward lookups, while XLOOKUP is more flexible. However, VLOOKUP remains essential for legacy workbooks and simplicity.

✅ Always use FALSE as range_lookup to avoid incorrect matches.
