XLOOKUP Tutorial – Next-Gen Lookup

XLOOKUP is the modern replacement for VLOOKUP/HLOOKUP. It's more flexible and intuitive.

=XLOOKUP(lookup_value, lookup_array, return_array, [if_not_found], [match_mode], [search_mode])

✨ Example: =XLOOKUP("Kurt", A2:A100, B2:B100) returns the matching value instantly.
