📘 Excel Macros & VBA Tutorial

📌 What are Excel Macros?

Macros are small programs written in VBA (Visual Basic for Applications) that automate repetitive tasks in Excel. With a single click, you can execute complex sequences — from formatting data to generating reports.

Think of macros as your personal Excel assistant: record actions or write custom logic to save hours of manual work.

VBA (Visual Basic for Applications) is the programming language behind macros, giving you full control over Excel objects, ranges, and events.

⚡ Why Use Macros?

Save time: Automate daily routines like data cleaning, formatting, and report generation.

Eliminate human errors: Standardize processes and avoid manual mistakes.

Boost efficiency: Combine multiple steps into one-click automation.

Complex logic: Create custom functions, user forms, and integrate with other Office apps.

Scalability: Process thousands of rows in seconds with loops and arrays.

🛠️ VBA Macro Example: Auto-Format & Summary

This macro scans the active sheet, applies professional formatting to a selected table, and inserts a summary row with totals. Perfect for sales or inventory sheets.

Sub AutoFormatAndTotal()
    Dim lastRow As Long
    Dim dataRange As Range
    
    ' Find last row with data in column A
    lastRow = Cells(Rows.Count, 1).End(xlUp).Row
    
    ' Set range from A1 to last column and row
    Set dataRange = Range("A1").CurrentRegion
    
    ' Apply border and header style
    With dataRange
        .Borders.LineStyle = xlContinuous
        .Rows(1).Font.Bold = True
        .Rows(1).Interior.Color = RGB(68, 114, 196)
        .Rows(1).Font.Color = vbWhite
    End With
    
    ' Add total row (simple sum for numeric columns)
    Dim col As Integer
    For col = 2 To dataRange.Columns.Count
        If Application.WorksheetFunction.IsNumber(dataRange.Columns(col).Cells(2)) Then
            Cells(lastRow + 1, col).Formula = "=SUM(" & dataRange.Columns(col).Address & ")"
            Cells(lastRow + 1, col).Font.Bold = True
        End If
    Next col
    
    Cells(lastRow + 1, 1).Value = "GRAND TOTAL"
    Cells(lastRow + 1, 1).Font.Bold = True
    
    MsgBox "✅ Formatting complete! Macro finished.", vbInformation, "Excel VBA"
End Sub

How to use: Press Alt + F11 to open VBA editor, insert a module, paste the code, and run by pressing F5 or assign to a button.

⭐ You can also record a macro (Developer tab → Record Macro) and then edit the VBA code for advanced customizations.

🛡️ Macro Security & Best Practices

Enable macros safely: Always save macro-enabled workbooks as .xlsm format. Go to File → Options → Trust Center → Trust Center Settings → Enable all macros (with caution) or digitally sign your projects.

Best practices: Avoid using .Select / .Activate, use Application.ScreenUpdating = False for speed, and always include error handling (On Error Resume Next or structured handlers).

Personal Macro Workbook: Store macros in PERSONAL.XLSB to make them available across all Excel files.
