XLOOKUP Tutorial

What is XLOOKUP?

XLOOKUP is a powerful Excel formula used to search for a value in a range and return a corresponding result from another column or row. It replaces older formulas like VLOOKUP and HLOOKUP.

Why Use XLOOKUP?

✔ Search left or right (unlike VLOOKUP)

✔ No need for column numbers

✔ Handles errors easily

✔ More flexible and powerful

Formula Syntax

=XLOOKUP(lookup_value, lookup_array, return_array)

Example

Find the salary of an employee:

=XLOOKUP(A2, A:A, B:B)
