VLOOKUP Tutorial

What is VLOOKUP?

VLOOKUP is used to search for a value in the first column of a table and return a corresponding value from another column.

When to Use

✔ Searching data in tables

✔ Matching IDs or names

✔ Simple lookup tasks

Formula Syntax

=VLOOKUP(value, table, column_index, FALSE)

Example

=VLOOKUP(A2, A:B, 2, FALSE)
